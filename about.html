The Footnotify Story

Footnotify is an interactive web based football resource for the premier league, it contain live streaming, score updates, league news, league table, and team data all in one place. That too personalized for each user. But the main idea is to get all these useful things in one place, people usually need to go to different websites for all the stuff, for example, a person will use onefootball (an android based app) for live scores, footyroom (a website) for highlights and any other website for live streaming the matches. But with Footnotify we will get all that stuff in one place. That too with personalization that is the news, highlights and data will be shown according to the person’s chosen team, so that they only see what they want to see.
